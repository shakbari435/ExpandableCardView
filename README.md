# Usage




### Step 1 
- Add it in your root build.gradle at the end of repositories:
```java
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
}
```



### Step 2 
- Add the dependency
```java
  	dependencies {
		    implementation 'com.github.shakbari435:ExpandableCardView:1.0'
	}
}
```
